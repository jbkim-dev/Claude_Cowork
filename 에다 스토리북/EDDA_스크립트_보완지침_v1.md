## 📋 발견된 오류 전체 목록 (EDDA #001 기준)

### 오류 1 — 인과관계 오류 (87~88줄)
> "이미르는 먹어야 했다. 그래서 아우둠블라가 나타났다."

**문제:** "먹어야 했기 때문에 나타났다"는 서술은 아우둠블라가 이미르의 필요에 의해 소환된 것처럼 읽힌다.  
**원전:** 아우둠블라는 이미르와 동시에, 같은 원시 얼음(흐림투르/Hrimfaxi)에서 태어났다. 이미르를 먹이기 위해 태어난 게 아니라 두 생명이 함께 탄생했고, 결과적으로 아우둠블라가 이미르를 먹였다.  
**규칙:** → **[규칙 1] 인과관계 원전 검증 원칙** 참조

### 오류 2 — 캐릭터 역할 단절 (89~101줄)
> 아우둠블라 등장 → 이미르에게 젖을 먹이는 장면 완전 생략 → 곧바로 얼음 핥기 장면

**문제:** 아우둠블라에게는 두 가지 역할이 있다.  
① 이미르에게 젖을 먹이는 역할  
② 소금 얼음을 핥아 부리(Búri)를 꺼내는 역할  
역할 ①이 완전히 빠진 채 역할 ②만 서술되어, 앞서 "이미르는 먹어야 했다"는 설정이 허공에 뜨게 된다.  
**규칙:** → **[규칙 2] 복수 역할 캐릭터 완결 서술 원칙** 참조

### 오류 3 — 계보 생략 (100~106줄)
> "그가 부리였다. 최초의 신." → (다음 줄) "그리고 신들이 나타났다. 오딘. 빌리. 베."

**문제:** 부리에서 오딘까지는 2세대가 필요하다.  
부리 → 아들 보르(Borr) → 보르 + 거인족 딸 베스틀라(Bestla) = 오딘·빌리·베  
이 계보를 생략하면 부리가 곧 오딘인 것처럼 오해된다.  
**규칙:** → **[규칙 3] 계보·인과 연결고리 완결 원칙** 참조

---

### 오류 4 — 화자 불명확한 대화 (117~119줄)
> "저들은 증가한다." / "저들은 강해진다." / "우리는 사라진다."

**문제:**  
① 따옴표 대화인데 화자가 누구인지 앞 문맥에서 전혀 확립되지 않음.  
② 스칼드의 나레이션이라면 따옴표를 쓰면 안 됨.  
③ 프로젝트 지침 "심리 묘사 금지"와 충돌 — 신들의 내면적 판단을 직접 표현.  
**규칙:** → **[규칙 4] 대화 화자 명확화 원칙** 참조

### 오류 5 — 복선 없는 속성 사용 (138~140줄)

**문제:** 앞선 90여 줄 어디에도 이미르가 "불멸처럼 보인다"는 복선이 없다. 갑자기 등장하는 속성이라 공허하게 느껴진다.

### 오류 6 — 주체 불명확 (132~135줄)
> "신들이 이미르에게로 나아갔다."

**문제:** 이미르를 죽인 것은 "신들(복수)"이 아니라 오딘·빌리·베, 세 명이다. 106줄에서 이미 세 신을 소개했으므로 "신들이"가 아닌 "오딘, 빌리, 베가" 또는 "세 신이"가 정확하다.  
**규칙:** → **[규칙 6] 주어 정확성 원칙** 참조

**문제:** "부서졌다"의 주어가 불명확하다. 뇌가 부서진 건지, 다른 무언가가 부서진 건지 앞 문맥만으로는 알 수 없다.

**문제:** "갈라졌다"의 주어가 없다. 긴눙가갑(태초의 허공)이 갈라진 건지, 무언가가 갈라진 건지 불명확하다.

### 오류 10 — 인간 창조 원전 왜곡 (180줄)
> "파도가 두 나무를 밀어왔다."

**문제:** 원전에서 오딘 일행이 발견한 두 나무(물푸레나무 아스크, 느릅나무 엠블라)는 해변에 쓰러져 있었고, "파도가 밀어온" 것이 아니다. 또한 인간에게 각기 다른 것을 부여한 세 신의 역할(오딘: 생명의 숨결, 빌리: 이성·감각, 베: 혈색·언어)이 생략되어 인간 창조가 단순화됐다.

## 📏 EDDA 스크립트 보완 규칙 (전 에피소드 적용)

### [규칙 1] 인과관계 원전 검증 원칙
> **"A이기 때문에 B가 나타났다/일어났다"는 서술은 반드시 원전 확인 후 사용한다.**

- "A와 B가 함께 일어났다"와 "A 때문에 B가 일어났다"는 완전히 다른 서술이다.
- 인과 연결어("그래서", "때문에", "그 결과")를 쓰기 전, 원전에 해당 인과관계가 실제로 있는지 검증한다.
- 원전에 인과가 명시되지 않은 경우, 시간순 병렬 서술("그리고", "한편", "같은 시간")로 대체한다.

**적용 예:**
```
❌ "이미르는 먹어야 했다. 그래서 아우둠블라가 나타났다."
✅ "같은 얼음에서 또 다른 생명이 태어났다. 거대한 암소, 아우둠블라."
   "아우둠블라의 젖 네 줄기가 흘렀다. 이미르는 그것으로 살았다."
```

---

### [규칙 2] 복수 역할 캐릭터 완결 서술 원칙
> **한 캐릭터가 두 가지 이상의 역할을 할 경우, 각 역할을 등장 순서대로 완결하여 서술한다.**

- 역할 A를 소개한 뒤 바로 역할 B로 넘어가면 역할 A가 공허해진다.
- 역할 A를 짧게라도 완결하고 전환 문장을 쓴 뒤 역할 B로 넘어간다.
- 전환 문장 예시: "한편", "그리고 아우둠블라는 또 다른 일을 했다."

**적용 예:**
```
아우둠블라의 역할 A (이미르에게 젖을 먹임) → 완결
→ 전환: "한편, 아우둠블라는 소금 얼음을 핥기 시작했다."
아우둠블라의 역할 B (부리를 꺼냄) → 서술
```

---

### [규칙 3] 계보·인과 연결고리 완결 원칙
> **신화 속 A→B→C 계보나 인과 흐름에서 중간 단계를 생략하지 않는다.**

- 계보는 최소 한 줄로라도 연결한다.
- 특히 신들의 혈통, 사건의 발단→전개, 저주·예언의 실현 과정은 반드시 연결 고리를 명시한다.

**적용 예:**
```
❌ "부리였다. (다음 줄) 오딘. 빌리. 베."
✅ "부리의 피에서 보르가 태어났다."
   "보르는 거인족의 딸 베스틀라와 맺어졌다."
   "그 사이에서 셋이 태어났다. 오딘. 빌리. 베."
```

---

### [규칙 4] 대화 화자 명확화 원칙
> **따옴표 대화 또는 인용구를 쓰기 전, 앞 문장에서 화자를 반드시 확립한다.**

- 스칼드의 나레이션은 따옴표 없이 서술한다.
- 신들의 대화를 인용하려면 "오딘이 말했다" 또는 연출 마커에서 명확히 표기한다.
- 심리나 내면의 판단을 직접 따옴표 대화로 쓰지 않는다 (프로젝트 지침 "심리 묘사 금지" 준수).

**적용 예:**
```
❌ (화자 없이) "저들은 증가한다." / "우리는 사라진다."
✅ (나레이션으로 전환)
   "거인족이 멈추지 않았다."
   "세 신은 알고 있었다."
   "이대로라면 신들의 자리는 없다."
```

---

### [규칙 5] 복선 없는 속성 사용 금지 원칙
> **캐릭터의 특성(불멸, 최강, 전능 등)을 클라이맥스에서 갑자기 사용하려면, 반드시 앞에서 복선을 심는다.**

- 해당 속성을 처음 서술할 때는 묘사 형태로 자연스럽게 확립한다.
- 클라이맥스에서는 복선 회수 형태로 사용한다.

**적용 예:**
```
[복선 심기 — 이미르 등장 시]
"이미르는 세상보다 먼저 있었다."
"그는 세상 그 자체였다."
"그를 멈추는 것은 불가능해 보였다."

[회수 — 클라이맥스]
"세상보다 먼저 있었던 이미르가 쓰러졌다."
```

---

### [규칙 6] 주어 정확성 원칙
> **모든 문장에서 행동의 주체를 명확하게 한다.**

- 리듬을 위해 주어를 생략할 때는, 앞 문장의 주어가 그대로 이어지는 경우에만 허용한다.
- 주어가 바뀔 때는 반드시 명시한다.
- "갈라졌다", "부서졌다" 같은 자동사는 주어 생략 시 의미가 불명확해지므로 주어를 쓴다.

**적용 예:**
```
❌ "뇌를 던졌다. 부서졌다. 구름이 되었다."
✅ "뇌를 던졌다. 뇌가 산산이 부서졌다. 구름이 되었다."

❌ "처음에는 아무것도 없었다. 그러다 갈라졌다."
✅ "처음에는 긴눙가갑이 있었다. 아무것도 없는 허공이."
   "그 허공의 끝에서, 세상이 시작되었다."
```

### [규칙 7] 원전 핵심 디테일 누락 금지 원칙
> **에피소드의 주제와 직접 관련된 원전의 충격적 디테일은 반드시 포함한다.**

- 프로젝트 지침 "원전의 충격적 디테일" 항목을 스크립트 작성 전에 먼저 목록화한다.
- 목록화된 디테일 중 에피소드 맥락에 들어맞는 것은 생략하지 않는다.
- 단, 스크립트 흐름을 방해하는 경우 씬 마커나 나레이션 한 줄로 짧게라도 언급한다.

**EDDA #001 적용 사례:**
```
[반드시 포함해야 했던 원전 디테일]
- 베르겔미르(Bergelmir)와 아내의 생존 → 현재 거인족의 기원
- 인간 창조 시 세 신의 각기 다른 역할 (오딘: 숨결, 빌리: 이성, 베: 혈색·언어)
- 무스펠헤임의 불씨 → 별과 태양의 기원
```

---

### [규칙 8] 감정과 행동 묘사 대체 원칙 (기존 지침 보완)
> **심리나 감정은 반드시 행동·시각 묘사로만 표현한다. 단, 스칼드의 해석적 나레이션은 허용한다.**

프로젝트 지침의 "심리 묘사 금지"를 더 구체화한 규칙이다:

| 금지 | 허용 |
|------|------|
| "오딘은 두려웠다" | "오딘의 손이 멈췄다" |
| "로키는 화났다" | "로키의 눈이 타올랐다" |
| "신들은 결심했다" | "세 신의 시선이 이미르에게 향했다. 말이 필요 없었다." |
| "저들은 증가한다" (화자 불명 내면) | "거인족이 멈추지 않았다. 세 신은 알고 있었다." |
